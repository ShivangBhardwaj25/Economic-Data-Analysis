# Economic-Data-Analysis
Exploratory data analysis of global cost of living with affordability index and correlation insights using Python.
# 📊 Economic Data Analysis Project

## 📌 Overview

This project focuses on analyzing global cost of living data to understand economic differences across countries. It explores how various factors such as rent, groceries, and purchasing power influence the overall affordability of living.

---

## 🎯 Objective

The objective of this project is to analyze cost of living data across different countries and identify patterns in economic indicators that affect affordability.

---

## ❓ Problem Statement

Cost of living varies significantly across countries, but it does not directly indicate how affordable a country is. This project aims to analyze the relationship between cost of living and purchasing power to better understand real-world affordability.

---

## 📁 Dataset Description

The dataset contains the following columns:

* Rank
* Country
* Cost of Living Index
* Rent Index
* Cost of Living Plus Rent Index
* Groceries Index
* Restaurant Price Index
* Local Purchasing Power Index

These indicators help compare economic conditions across countries.

---

## 🛠 Tools & Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Jupyter Notebook

---

## 🧹 Data Cleaning & Preparation

* Removed duplicate records
* Cleaned column names for consistency
* Checked for missing values
* Converted column names into a standard format

---

## 📊 Exploratory Data Analysis (EDA)

### 🔹 Country-Level Analysis

* Identified top 10 most expensive countries
* Identified top 10 cheapest countries

### 🔹 Visualizations Used

* Bar Charts → Country comparison
* Scatter Plots → Relationship analysis
* Histogram → Distribution analysis
* Box Plot → Outlier detection

---

## 🔥 Advanced Analysis

### 📌 Affordability Index (Custom Feature)

A new feature was created to measure real affordability:

Affordability Index = Local Purchasing Power Index / Cost of Living Index

This helps identify countries that are truly affordable, not just cheap.

---

### 📌 Correlation Analysis

* Performed correlation analysis between all numeric variables
* Used heatmap to visualize relationships
* Found strong correlation between cost of living, rent, and groceries

---

## 📊 Key Insights

* Cost of living is highly influenced by rent and grocery prices
* Countries with high cost often also have high purchasing power
* Some countries appear cheap but are less affordable due to low income levels

---

## 📖 Storytelling Insight

The analysis shows that cost of living alone does not determine affordability.

Some countries with high living costs also offer high purchasing power, making them relatively affordable for residents. On the other hand, countries with low costs may still be less affordable due to lower income levels.

This highlights the importance of considering both income and expenses when evaluating economic conditions.

---

## 🏁 Conclusion

* Cost of living varies significantly across countries
* Rent and groceries are major contributors to high living costs
* Purchasing power is essential for understanding real affordability
* A combined approach provides better economic insights

---

## ⚠️ Limitations

* Dataset does not include income distribution data
* No time-based trends available
* External economic factors like inflation are not considered
